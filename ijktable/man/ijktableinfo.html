NAME

     ijktableinfo - output isosurface table information

SYNOPSIS

     ijktableinfo

DESCRIPTION

     ijktableinfo is a program to output isosurface table  infor-
     mation  about  the isosurface table and the isosurface table
     polyhedron.

OPTIONS

     -poly
          Print out a  complete  description  of  the  isosurface
          polyhedron including list of vertex coordinates, a list
          of edges and a list of facets.

     -stat
          Print out isosurface polyhedron  and  table  statistics
          including  number  of  polyhedron  vertices,  edges and
          facets, number of table entries,  max  number  of  sim-
          plices  per  entry  and average number of simplices per
          entry.  -stat is the default.

     -isovert
          Print out list of isosurface vertices.

     -isoedge
          Print out maximum  and  average  number  of  isosurface
          edges per entry.

     -entry num
          Print entry  num  from  the  isosurface  table.   Print
          number  of simplices in the entry and a list of simplex
          vertices.  Each number in the list is the index  of  an
          isosurface vertex.

     -h   Print a help message.
