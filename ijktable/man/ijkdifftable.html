NAME

     ijkdifftable -  output  the  difference  of  two  isosurface
     tables

SYNOPSIS

     ijkdifftable [-h] {file1} {file2}

DESCRIPTION

     ijkdifftable is  a  program  to  output  the  difference  in
     polyhedra  and  in  table  entries of two isosurface tables.
     ijkdifftable halts if the tables have  different  dimensions
     or  if  the  polyhedra have different vertices or edges.  It
     does not halt if the polyhedra have different facets, or the
     table  entries are different.  ijkdifftable ignores the sim-
     plex order in table entries and ignores the vertex order  in
     comparing isosurface simplices.

OPTIONS

     -h   Print help message (and exit.)

NOTES

     In comparing two isosurface simplices, the vertex orders may
     imply  different  simplex  orienations but ijkdifftable will
     not recognize this as a difference.
